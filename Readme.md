# Card Game

This game is coded as part of interview process at SAP AG. The game has been built using TypeScript (Node JS). Below dependency modules have been used while programing the game. Game output is displayed in the terminal.

 - @types/chai: "^4.2.14",
 - @types/mocha: "^8.2.0",
-    @types/node: "^14.14.20",
 -   chai: "^4.2.0",
  -  mocha: "^8.2.1",
   - ts-node: "^9.1.1",
-    typescript: "^4.1.3"

To install dependencies run below command.
```sh
$ npm install
```

Use below commands to Build, Run and Test the code.

### Build
```sh
$ npm run build
```

### Run
```sh
$ npm run start
```
### Test
```sh
$ npm run test
```
Unit tests have been written using Chai and Mocha.
